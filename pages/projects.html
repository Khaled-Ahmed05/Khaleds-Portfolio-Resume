WeatherForecast - NodeJS/Express

Weather Forecast Website

- Developed an information web application with location-based info about weather forecast.

- Users provide a location they want to inquire about its weather forcast.

- System sends the location to an external API that gets the coordinates to this location, then sends it to another external API that gets weather forecast information about said coordinates, and then displays it to the user.

- Implemented methods to tell the user if the location they entered is not valid.
